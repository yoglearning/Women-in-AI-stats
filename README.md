# Women & AI — The Numbers

An interactive stats constellation built for **The Great AI Adventure** blog.

**Live demo:** [View on GitHub Pages](https://yourusername.github.io/tgaia-women-ai-stats)

## About

This interactive tool accompanies the article **"Why Every Woman Should Care About the AI Revolution"** published on Medium for International Women's Day 2025.

12 interconnected bubbles, each holding a real statistic about women and AI. Click any bubble to explore the full story and navigate to connected data points.

**Categories covered:**
- 🌸 Risk & Displacement
- ✨ Workforce Gap
- 🌊 Momentum
- 💜 Confidence Gap
- 🔍 Hidden Bias

## Sources

- LinkedIn Economic Graph
- Taulbee Survey & IPEDS Data

## How to use

1. Open `index.html` in any browser
2. Or visit the GitHub Pages URL above
3. Click any bubble to expand the stat and see connected data points

## Built with

Pure HTML, CSS, and vanilla JavaScript. No dependencies. No frameworks.
